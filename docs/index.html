Features
Everything Your Clan Needs
Powerful, automated tools to keep your community organized, engaged, and growing.

Clan Tag Tracking

Real-time monitoring of clan tags across all members. Instantly detect who's missing or has the wrong tag with automated scanning.

Streak System

Daily check-in streaks with milestone rewards. Gamified tracking keeps your community active and engaged every day.

Live Dashboard

Web-based real-time dashboard with server statistics, member activity, and tag compliance — accessible from any browser.

Audit Log

Complete, timestamped history of every tag change. Full accountability with who changed what, and when.

Multi-language

Full Turkish and English support with i18n system. Serve global communities without any language barriers.

Auto Welcomer

Configurable welcome messages for new members with customizable templates. First impressions that count.

Setup
Up & Running in 3 Steps
Get your clan bot online in under 5 minutes.
1

Clone & Install

Clone the repository and install all dependencies.

git clone https://github.com/KynuxDev/Discord-Clan-Bot.git && cd Discord-Clan-Bot && npm install
2

Configure

Copy the environment template and add your Discord bot token.

cp .env.example .env && nano .env
3

Launch

Start with Node.js directly or deploy with Docker.

node src/index.js # or: docker compose up -d
