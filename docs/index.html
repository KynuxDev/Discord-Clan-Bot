Features
Built for serious
clan management.
Everything you need to run a disciplined, active, and well-organized Discord clan.
01
Clan Tag Tracking
Automated real-time monitoring across all server members. Detect missing or incorrect tags instantly with bulk scanning, and get notified when members change their usernames.
02
Streak System
Daily check-in tracking with milestone rewards. Gamified engagement that keeps members active.
03
Live Dashboard
Web-based real-time analytics. Monitor compliance rates, member activity, and tag history from any browser.
04
Audit Log
Timestamped records of every change. Full accountability — who changed what, and when.
05
Multi-language
Complete Turkish and English i18n support. Serve global communities natively.
06
Auto Welcomer
Configurable templates that greet new members. Custom embeds, role assignments, and first-touch messaging.

Setup
Running in 3 steps.
From zero to live bot in under 5 minutes.
01

Clone & Install

Pull the repository and install all dependencies.

git clone https://github.com/KynuxDev/Discord-Clan-Bot.git && npm install
02

Configure

Add your Discord bot token to the environment file.

cp .env.example .env
03

Launch

Start with Node.js or deploy with Docker.

node src/index.js # or: docker compose up -d

Start managing
your clan.
